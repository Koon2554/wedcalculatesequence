How to Use

How to Use the Website

1. Enter sequence numbers

  • The website has 2 input fields
  • First field is for entering 4-5 examples of the sequence you want to study
  • Input format: Use commas (,) to separate numbers, e.g. 2,3,4,5,6
  • Second field is for entering the term number of the sequence you want to find, e.g. 67

2. Process the sequence

  • After filling both fields, press the Calculate button
  • The system will show the requested term value with step-by-step calculation and explanation

3. Input format settings

  • Users can switch to the second format which creates separate input fields for each value
  • There is a Add input field button to add new fields, each field can only input 1 number value
  • After filling all fields, press the View entered values button
  • The website will show all entered data with commas (,) separating the numbers for users to verify correctness

About

About Us Finding the nth term of a mathematical sequence is an important topic in many fields, such as weather forecasting, data analysis, statistical calculations, and problem-solving in science and engineering. The problem is that manually calculating the nth term can be time-consuming and prone to errors, especially when the sequence's form is complex or requires high precision. Furthermore, understanding the sequence's form and analyzing the results are crucial for practical application. The team therefore aimed to study and develop a process for calculating the nth term for various sequences. They also designed and developed a website that can help users quickly and accurately calculate the nth term, complete with explanations for further study.
